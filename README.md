## 🎯 The Problem

> **70% of rural schools in Latin America have no science lab.**

Students in places like the Peruvian Amazon memorize formulas without ever experiencing the phenomenon behind them. A pendulum is just "T = 2π√(L/g)" — never an object they can swing and measure.

**Yachay Lab changes that.** Any student with a browser can now access a complete physics and chemistry laboratory — for free, with no ads, no registration required.

---

## ✨ What Yachay Lab Does

| Feature                  | Description                                                                                                      | Technology                     |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------- | ------------------------------ |
| ⚗️ **17 Simulators**     | Physics & chemistry in real time — Projectile Motion, Pendulum, Ohm's Law, Ideal Gases, Statics I & II, and more | Canvas API + Euler integration |
| 🤖 **AI Tutor (Yachay)** | Socratic AI that guides with questions, never direct answers. Uses Amazon Rainforest analogies                   | Groq + LLaMA 3.3 70B           |
| 🎯 **Adaptive Practice** | Elo rating system (K=32) that adjusts difficulty per topic automatically                                         | Custom Elo algorithm           |
| ⚔️ **STEM Battle**       | Real-time 1v1 multiplayer quiz against friends via room codes                                                    | Supabase Realtime (ready)      |
| 🔥 **Streak System**     | Daily streak tracker with milestones, XP, confetti and ring animation                                            | localStorage + Canvas          |
| 🏆 **Global Ranking**    | Live leaderboard with Elo tiers, flags, and filters                                                              | Shared localStorage → Supabase |
| 🎖️ **30 Achievements**   | Unlockable badges across 4 categories with XP rewards                                                            | Custom achievement engine      |
| ⚡ **Daily Missions**    | 4 daily missions generated by seed — renew at midnight                                                           | Seeded PRNG                    |
| 🎤 **Voice Input**       | Speak your questions to Yachay AI — Web Speech API with volume control                                           | Web Speech API                 |
| 🌍 **Guest Geo ID**      | Guests get a unique identity based on their real IP, city, country and device fingerprint                        | SHA-256 + ipapi.co             |
| 📅 **Season Challenges** | Monthly themed challenge with progress bar and exclusive badge                                                   | localStorage                   |

## 🔑 Activating Supabase (for real-time multiplayer)

1. Create a free project at [supabase.com](https://supabase.com)
2. Run this SQL in the editor:

```sql
create table battles (
  id text primary key,
  host_id text, guest_id text,
  host_name text, guest_name text,
  topic text, rounds int,
  status text default 'waiting',
  questions jsonb, scores jsonb default '{"host":0,"guest":0}',
  created_at timestamptz default now()
);
alter table battles enable row level security;
create policy "public" on battles for all using (true);
```

3. In `battle.js`, set `const SUPABASE_ACTIVE = true` and uncomment the Supabase import block.

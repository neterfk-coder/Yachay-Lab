## 🎯 The Problem

> **Millions of students around the world study science without ever performing a real experiment.**

In schools with limited resources, students memorize formulas like `T = 2π√(L/g)` without ever swinging a real pendulum. They write `F = ma` on exams without ever measuring acceleration. STEM becomes an abstract language disconnected from reality.

**Yachay Lab changes that.** Any student with a browser can now access a complete physics and chemistry laboratory — for free, with no installation and no registration required.

---

## ✨ Features

### ⚗️ 17 Interactive Simulators

Real-time physics and chemistry experiments powered by the **Canvas API with Euler integration** — no external physics library.

| Area | Simulators |
|------|-----------|
| ⚡ **Physics** | URM, MRUA, Free Fall, Projectile Motion, Simple Pendulum, Hooke's Law, Waves, Ohm's Law, Optics, Mechanical Energy, Statics I, Statics II |
| 🧪 **Chemistry** | pH & Acid-Base, Ideal Gases, Chemical Reactions, Titration, Periodic Table |

**Exercise Input Panel** — Students enter their own exercise values, click **"Load into simulator"**, and press **▶ Start** to see their specific problem animated in real time.

---

### 🤖 Yachay AI Tutor

Powered by **Groq + LLaMA 3.3 70B Versatile** (700+ tokens/sec).

- Uses the **Socratic method** — never gives the direct answer, guides with questions
- Students can **type or speak** via Web Speech API
- Voice synthesis — Yachay speaks the responses aloud
- Amazon Rainforest analogies to make concepts relatable

---

### 🎯 Adaptive Practice with Elo

$$E = \frac{1}{1 + 10^{(1000 - \text{Elo})/400}}$$

- Elo rating system (K=32) per topic — Physics, Chemistry, Math
- Automatically adjusts difficulty based on performance
- Historical Elo progress chart (Chart.js)
- 30 questions across 3 areas at intermediate-advanced level

---

### ⚔️ Real-time STEM Battles

- 1v1 multiplayer via **6-character room code**
- Synchronized with **Supabase Realtime** channels
- 5-10 rounds, 15 seconds per question
- Live rival status updates

---

### 📝 AI Exam Generator

- Groq generates personalized exams based on current Elo level
- Auto-graded with step-by-step explanations
- Downloadable **PNG certificate** generated with Canvas API
- Shareable on WhatsApp and social media

---

### 🏆 Full Gamification Engine

- **Global ranking** with live Elo tiers (Beginner → Bronze → Silver → Gold → Platinum → Diamond)
- **30 achievements** across 4 categories
- **20-mission daily pool** generated by seeded PRNG — renewed at midnight
- **Monthly season challenges** with exclusive badge and progress bar
- **Streak system** with 6 milestones and XP rewards
- **Daily push notifications** for streak reminders

## 🚀 Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/neterfk-coder/Yachay-Lab.git
cd Yachay-Lab

# 2. Add your Groq API key (free at console.groq.com)
# Create config/keys.js (already in .gitignore):
echo "window.GROQ_KEY_LOCAL = 'gsk_your_key_here';" > config/keys.js

# 3. Open in browser
# Just open index.html — no build step, no npm install
```

**No build step. No dependencies. No npm install.** Open `index.html` directly.

---

## 🗄️ Database Setup (Supabase)

1. Create a free project at [supabase.com](https://supabase.com)
2. Run `supabase-setup.sql` in the SQL Editor
3. Add environment variables in Vercel:

```
GROQ_KEY      = gsk_your_groq_key
SUPABASE_URL  = https://your-project.supabase.co
SUPABASE_KEY  = your_anon_key
```

The `build.js` script generates `config/keys.js` automatically from Vercel secrets on every deploy.

---

## 🔐 Security

- API keys are **never committed** to GitHub
- `config/keys.js` is in `.gitignore`
- Vercel environment variables are used in production
- Supabase Row Level Security enabled on all tables
- Guest identity uses SHA-256 fingerprinting — no PII stored
